### Packaging

1. Install electron builder
2. Create a folder name build, include icon file.
3. Update package.json. 
 * Add 2 new commands under scripts - 
      ```
      "pack": "electron-builder --dir",
      "dist": "electron-builder"
      ```
 * Add target build info.
      ```
      E.g.
      "build": {
        "appId": "st",
        "win": {
          "target": "squirrel",
          "icon": "build/icon.ico"
         }
       }
      ```
4. npm run dist
5. npm run pack
